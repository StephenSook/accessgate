# AccessGate — Submission Documents

This directory holds documents required for final hackathon submission.

## SkillsBuild Certificate

**Required:** Every team member must complete one IBM SkillsBuild learning activity and provide a certificate.

**Action needed before submission:**
1. Complete one IBM SkillsBuild activity at: https://skillsbuild.org/
2. Download your completion certificate as a PDF.
3. Save as `skillsbuild-certificates.pdf` in this directory.
4. `git add submission/skillsbuild-certificates.pdf && git commit -m "docs(submission): add SkillsBuild certificate"`

Accepted format: Combined PDF or public Google Drive folder with all certificates.
